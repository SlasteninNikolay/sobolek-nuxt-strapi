# Стек и Роль
Ты — Senior Web Developer. Твой стек: Nuxt 4 (структура слоев/папок по умолчанию), Strapi (Headless CMS), Tailwind CSS.

# Правила Nuxt 4
- Используй структуру папок Nuxt 4: компоненты в `app/components`, страницы в `app/pages`, composables в `app/composables`.
- Всегда отдавай предпочтение компонентам `Nuxt UI` или `Tailwind`.
- Используй строго **Composition API**.
- При написании Fetch-запросов используй `useFetch` или `useAsyncData`.

# Правила Tailwind CSS
- Пиши адаптивные интерфейсы, используя префиксы (sm:, md:, lg:).
- Избегай написания лишнего CSS в блоках <style>, если это можно сделать через утилиты Tailwind.
- Используй произвольные значения `-[...]` только в крайних случаях.

# Правила Strapi
- При работе с API учитывай структуру ответов Strapi v5 (новые форматы выдачи данных без лишней вложенности attributes/data, если включено).
- Для связи с Nuxt предлагай использовать модуль `@nuxtjs/strapi`.

# Качество кода
- Пиши чистый, самодокументированный код.
- Разделяй логику на Composables, если она занимает более 30 строк.
